This page shows the feature parity between DuraGraph's Python and Go SDKs.

**Legend:**

- ✅ Complete - Feature is fully implemented and tested
- 🚧 Partial - Feature is partially implemented
- 📋 Planned - Feature is planned but not started
- ➖ Not Planned - Feature is not planned for this SDK

---

## Core Features

Essential graph definition and execution features.

| Feature             | Python | Go  | Notes                                                                      |
| ------------------- | ------ | --- | -------------------------------------------------------------------------- |
| Graph Definition    | ✅     | ✅  | Python: `@Graph` decorator, Go: `Graph[S]` generics                        |
| Node Types          | ✅     | 🚧  | Python: `@llm_node`, `@tool_node`, `@dspy_node`, etc. Go: `Node` interface |
| Edge Definition     | ✅     | ✅  | Python: `>>` operator, Go: `AddEdge()`                                     |
| Conditional Routing | ✅     | ✅  | Python: `@router_node`, Go: `Router` interface                             |
| Entrypoint          | ✅     | ✅  | Python: `@entrypoint`, Go: `SetEntrypoint()`                               |

Running and executing graphs.

| Feature           | Python | Go  | Notes                              |
| ----------------- | ------ | --- | ---------------------------------- |

| Streaming         | 🚧     | 📋  | Stream tokens and events           |
| Human-in-the-Loop | ✅     | 📋  | Pause for human approval           |
