The Go SDK is currently in development. For now, you can interact with DuraGraph using the REST API.

## Using the REST API

```go
package main

import (
    "bytes"
    "encoding/json"
    "fmt"
    "net/http"
)

const baseURL = "http://localhost:8081/api/v1"

func main() {
    // Create an assistant
    assistant := map[string]interface{}{
        "name":         "my-assistant",
        "graph_schema": map[string]interface{}{},
    }

    body, _ := json.Marshal(assistant)
    resp, err := http.Post(baseURL+"/assistants", "application/json", bytes.NewBuffer(body))
    if err != nil {
        panic(err)
    }
    defer resp.Body.Close()

    var result map[string]interface{}
    json.NewDecoder(resp.Body).Decode(&result)
    fmt.Printf("Created assistant: %v\n", result)
}
```

## LangGraph SDK Compatibility

You can also use any LangGraph-compatible Go client, as DuraGraph implements the same API:

```go
// Point your LangGraph client to DuraGraph
client := langgraph.NewClient("http://localhost:8081")
```
